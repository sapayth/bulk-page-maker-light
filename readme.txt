== Installation ==

= Minimum Requirements =

* PHP 7.4 or higher
* MySQL 5.1 or higher

= Automatic installation =

1. Log in to your WordPress dashboard.
2. Go to Plugins > Add New.
3. Search for "Bulk Page Maker Light".
4. Click "Install Now" and then "Activate".

= Manual installation =

1. Download the Bulk Page Maker Light plugin.
2. Upload the plugin folder to your `/wp-content/plugins/` directory via FTP.
3. Activate the plugin through the Plugins menu in WordPress.

= Updating =

Automatic updates are supported. Always back up your site before updating.

== Frequently Asked Questions ==

= Is there a limit to how many pages or posts I can create? =
No! You can create as many as you need, limited only by your server resources.

= Does this plugin work with custom post types? =
Currently, Bulk Page Maker Light supports Pages and Posts. Support for custom post types is coming soon!

= Will this slow down my website? =
No, Bulk Page Maker Light is optimized for performance and built to WordPress standards.
